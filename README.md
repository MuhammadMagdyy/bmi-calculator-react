# ⚖️ BMI Master | React Precision Calculator

A professional, high-performance BMI (Body Mass Index) calculator built with **React 18** and **Vite**. This application features a refined dark-mode UI, support for dual-unit systems, and a persistent calculation history.

## 🚀 Key Features
- **Dual Unit Logic:** Toggle seamlessly between Metric (cm/kg) and Imperial (in/lbs) with automatic state reset.
- **Dynamic Health Categorization:** Real-time analysis of BMI data into four medical categories with color-coded UI feedback.
- **Data Persistence:** Integrated history log to track the last 5 calculations.
- **Modern UI/UX:** Built using Flexbox for a responsive, balanced layout and a custom CSS "Glassmorphism" aesthetic.

## 🛠️ Technical Stack

- **Build Tool:** Vite (for optimized HMR and production bundling)
